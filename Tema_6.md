## Лабораторная работа №5
### Написать функцию, которая будет сортировать кортеж, состоящий из целых чисел по возрастанию и возвращать его. Если хотя бы один элемент не является целым числом, тофункция возвращает исходный кортеж.

```python
def tsort(tuple1):
    for elem in tuple1:
        if not isinstance(elem, int):
            return tuple1
    return tuple(sorted(tuple1))

if __name__ == "__main__":
```

### Написать программу, которая будет показывать в каком порядке сотрудники входили и выходили из офиса. Написать функцию, которая на вход принимает кортеж и случайный элемент (id). Требуется вернуть новый кортеж, начинающийся с первого появления в нем и заканчивающийся вторым его появлением включительно.

```python
def func(tp, id):
    a, b = -1, -1
    for i in range(len(tp)):
        if tp[i] == id:
            if a != -1:
                b = i
            else:
                a = i
    return tp[a:b]


if __name__ == "__main__":
    print(func((2, 1, 8, 8, 5, 2, 6, 1, 6, 7, 6, 4, 0), 1))
```

## Самостоятельная работа №5
### Напишите программу, которая переносит все ненулевые числа в левую часть списка, а все нули - в правую. Задача должна быть решена за один проход по списку.

```python
def move_zeros(new_list):
    c = 0
    for i in range(len(new_list)):
        if new_list[i] != 0:
            new_list[c] = new_list[i]
            c += 1
    for k in range(c, len(new_list)):
        new_list[k] = 0
    return new_list


if __name__ == "__main__":
    my_list = [2, 0, 6, 0, 4, 3, 1, 4, 7, 0, 8, 2, 8]
    print(move_zeros(my_list))
```
